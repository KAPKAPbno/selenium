# Selenium AI Bot

Telegram-бот на базе нейросетей NVIDIA.

## Как запустить:

1. Склонируйте репозиторий:
`git clone https://github.com/KAPKAPbno/selenium.git`

2. Создайте виртуальное окружение и активируйте его.

3. Установите зависимости:
`pip install -r requirements.txt`

4. Скопируйте файл `.env.example` и назовите его `.env`. Впишите туда свои ключи API:
`TELEGRAM_TOKEN` (от BotFather)
`NVIDIA_API_KEY` (от NVIDIA API)

5. Запустите бота:
`python main.py`
